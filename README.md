This is the [GitHub repository](https://github.com/mrwatson-de/fmworkmate.github.io) for the [fmworkmate.com] website. 

fmworkmate.com is made by [mrwatson.de] and is the homepage of [fmWorkMate] and [MrWatson's Tools].  

fmworkmate.com is built with [Jekyll] and the [Just the Docs] theme and is published with [GitHub Pages]. 

In the repository you see the raw code of the website pages, but if you want to see the website and not the raw code,
just head on over to [fmworkmate.com].

Enjoy!
