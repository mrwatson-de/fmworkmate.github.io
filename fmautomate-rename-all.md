{{page.strapline}}

1. Select your code
2. Press <kbd>CMD</kbd><kbd>⌥</kbd><kbd>A</kbd> - for Replace **A**ll
3. Enter search text and replace text
4. Press `[Replace]`

Done!

## Multiple Replacements

If you want to do multiple replacements, just press `[Replace…]` to keep the dialog open.
