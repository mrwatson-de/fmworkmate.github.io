## File Authorisation problems?

Are you suffering from File Authorisation problems?

It may well be that you have duplicate File-IDs in your FileMaker solution. With fmMetaMate, you can easily check for duplicate File-IDs by saving a copy of your solution as XML and then checking the File-IDs in the first line of the XML files.

{: .note}
See the post on the [Claris Community](https://community.claris.com/en/s/question/0D7Vy00000IMiODKA1/detail) for more details on how to do this.

This will be getting easier sometime (😔) in the future.
