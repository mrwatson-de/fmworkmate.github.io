# {{page.title}}

![Mac](assets/images/mac.png){: .float-front-right .w-48}

fmSyntaxColorizer only works on MacOS systems.

To install fmSyntaxColorizer, simply

![fmSyntaxColorizer](fmsyntaxcolorizer.png){: .float-front-right .w-48}

1. Download the [latest release of fmSyntaxColorizer]({{page.github_latest}})
2. Unzip the archive
   - Note: If instead of using the latest release you have downloaded the latest code from the main repo using `Code > Download ZIP`, you will need to rename the `fmSyntaxColorizer-main` folder to just `fmSyntaxColorizer`
3. Move the `fmSyntaxColorizer` folder to your `Applications` folder (or wherever you like to keep your apps)
4. Double-click the `fmSyntaxColorizer.fmp12` file to launch fmSyntaxColorizer in FileMaker Pro

Once installed proceed to [fmSyntaxCoclorizer]
