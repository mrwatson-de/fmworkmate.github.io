# {{page.title}}

{{page.strapline}}

You can Transform your code in ways you would not believe possible.

Just

- Download, install and Setup  the [fmCheckMate XSLT Library](fmcheckmate-xslt-library.html)
- Convert your FileMaker code to XML in fmCheckMate
- Press `[T]` to transform

Choose from a wide range of transforms to analyse, change or transform your FileMaker code.
